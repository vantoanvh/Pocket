# Pocket

Pocket is a modern, beautifully animated backpack system for Roblox.

Unlike other backpack systems, Pocket is completely rewritten from the ground up. It is not a fork of Roblox’s default backpack. The design and internal structure are cleaner, more abstracted, and built without relying on deprecated methods.

The UI is inspired by [Satchel v2](https://github.com/ryanlua/satchel/tree/main), including its console-friendly design. Pocket also keeps a public API similar to Satchel, making it familiar for developers who have used Satchel before.

Pocket is designed to feel vanilla while being cleaner, more optimized, and easier to maintain.
